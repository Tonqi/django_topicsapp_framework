Django topics app framework
Basic web application with a User authentication system that lets you open threads to specific topics and comment/reply on them.

Install the requirements with: pip install -r requirements.txt (don't forget to create a venv)
Start the django app with: py manage.py runserver

For more information about user configuration e.g for the admin panel or other things, refer to the official django docs.
https://docs.djangoproject.com/en/6.0/
